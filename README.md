# pc-audit-windows
Engineer-grade Windows PC audit tool that outputs JSON + Markdown reports.


## Requirements

### Required
- Windows 10 / Windows 11
- Python 3.10+ (recommended: 3.11+)
  - During installation, enable: **Add Python to PATH**
